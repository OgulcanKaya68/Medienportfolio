# Kommunikations- & Medienportfolio

Digitales Portfolio für den Schulzweig **Medieninformatik** an der BHAK/BHAS Schwaz.

Die Website dokumentiert Projekte und Kompetenzen aus fünf Fachbereichen über mehrere Schuljahre hinweg — aufgebaut nach dem offiziellen Medienportfolio-Leitfaden der Schule.

## Fachbereiche

- **KOEA** — Kommunikation & Öffentlichkeitsarbeit (4.–5. Klasse)
- **MINF** — Medieninformatik (1.–3. Klasse)
- **IMSK** — Informationsmanagement & Medientechnik (2.–3. Klasse)
- **WOP** — Wirtschafts- & Organisationspsychologie (3.–5. Klasse)
- **NTMA** — Netzwerkmanagement (3.–4. Klasse)

## Technik

Statische Website — kein Framework, kein CMS.

- HTML, CSS, JavaScript
- Gehostet über Plesk (Schulserver)
